  <div className="text-center mb-12">
    <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">
      Quick Start
    </h2>
    <p className="text-base text-gray-600 dark:text-gray-400 max-w-2xl mx-auto">
      Get up and running in minutes with our interactive CLI or manual installation
    </p>
  </div>

  <Steps>
    <Step title="Install your preferred package">
      Choose the configuration package you need and install it with your package manager.

      <CodeGroup>
      ```bash pnpm
      # ESLint
      pnpm add -D @zayne-labs/eslint-config

      # Prettier
      pnpm add -D @zayne-labs/prettier-config

      # TypeScript
      pnpm add -D @zayne-labs/tsconfig
      ```

      ```bash npm
      # ESLint
      npm install -D @zayne-labs/eslint-config

      # Prettier
      npm install -D @zayne-labs/prettier-config

      # TypeScript
      npm install -D @zayne-labs/tsconfig
      ```

      ```bash yarn
      # ESLint
      yarn add -D @zayne-labs/eslint-config

      # Prettier
      yarn add -D @zayne-labs/prettier-config

      # TypeScript
      yarn add -D @zayne-labs/tsconfig
      ```
      </CodeGroup>
    </Step>

    <Step title="Configure your project">
      Use the interactive CLI for ESLint setup, or configure manually.

      <Tabs>
        <Tab title="ESLint (CLI)">
          Run the interactive setup wizard:
          ```bash
          pnpx @zayne-labs/eslint-config@latest
          ```
          The CLI will guide you through framework selection and dependency installation.
        </Tab>

        <Tab title="ESLint (Manual)">
          Create `eslint.config.js` in your project root:
          ```js
          import { zayne } from "@zayne-labs/eslint-config";

          export default zayne();
          ```
        </Tab>

        <Tab title="Prettier">
          Create `prettier.config.js` in your project root:
          ```js
          import { zayne } from "@zayne-labs/prettier-config";

          export default zayne({
            base: true,
            sortImports: true,
            tailwindcss: true,
          });
          ```
        </Tab>

        <Tab title="TypeScript">
          Add to your `tsconfig.json`:
          ```json
          {
            "extends": "@zayne-labs/tsconfig/bundler/dom/app"
          }
          ```
        </Tab>
      </Tabs>
    </Step>

    <Step title="Start linting and formatting">
      Add scripts to your `package.json`:

      ```json
      {
        "scripts": {
          "lint": "eslint .",
          "lint:fix": "eslint . --fix",
          "format": "prettier --write ."
        }
      }
      ```

      <Note>
        ESLint auto-detects TypeScript and React. Framework-specific configs like Vue, Svelte, and Astro require explicit enablement.
      </Note>
    </Step>
  </Steps>
</div>
